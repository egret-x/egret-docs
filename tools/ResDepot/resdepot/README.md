ResDepot是针对资源配置文件的创建，编辑和管理的工具。可以利用它来创建资源的九宫格数据，也可以对资源配置文件内指定的资源进行分组，发布等功能。发布即就是对资源配置文件内的资源进行合图，同时生成一个新的ResDepot文件，保证在项目代码不改变情况下减少资源数目。

ResDepot只是针对资源进行管理的工具，它自身没有项目的概念，它的管理单位是resource.json。

### 适应平台

* Windows
* Mac OS X
