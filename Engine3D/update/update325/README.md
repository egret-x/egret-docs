## 概述

在Egret3D 3.2.5版本中，修复了3.2.4版本中相应的Bug

### 更新内容

* 骨骼动画进行了大量改动,优化了骨骼动画的占用内存。使用3.2.5版本的引擎必须使用3.2.5版本的插件导出。
* 第2UV数量分离
* 不再支持 3DMax 插件导出。开发者使用3DMax做的模型，请先导入到Unity中，再通过 Unity 插件导出。

### Bug列表

* 修复多个物体事件bug
* 修复qq浏览器 Input 标签没有焦点的bug
* 修复插件导出点光源没有应用的bug
* 修复角色导出文件子点节重复加入bug
* 修复模型导出后顶点色bug
