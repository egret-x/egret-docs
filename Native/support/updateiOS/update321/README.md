## 更新内容

* [注意] 此版本添加了对渲染效率的优化，如果渲染时出现了异常请在引擎中关闭优化；
* [修复] 修复部分情况下渲染出错的问题；
* [修复] 修复从后台恢复时背景音乐没有继续播放的问题；
* [修复] 修复设置音频播放起始时间有延迟的问题；
* [改进] 提高稳定性。
